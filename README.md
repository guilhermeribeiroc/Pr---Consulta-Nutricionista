# Pre-Consulta - Nutricionista
Painel Clínico IA - Nutricionista Luma  Plataforma que integra n8n, Gemini IA e Streamlit para automatizar anamneses. Transforma dados do Tally em resumos clínicos e análise crítica de recordatório 24h via IA. Interface reativa em Python (Pandas) com visual em cards profissionais.
